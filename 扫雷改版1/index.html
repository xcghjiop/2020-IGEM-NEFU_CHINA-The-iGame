The rules of the game

1. Select different difficulty levels and press the "Start Game!" button to start the game.
2. Click the gray block and the number displayed is the number of mines in the eight squares around it.
3. You can determine the location of hidden mines by the number displayed and the color block depth (the deeper the more mines around), and right click to insert the red flag.
4. You need to place red flags on as many potential mines as possible in the shortest amount of time. ​
5. The game ends when you plant the last red flag or press the last gray square. Be sure to guess more mines before that.
6. Press the "Restart" button to start the game. Remember to click the "Start Game!" button to start the game timer.
